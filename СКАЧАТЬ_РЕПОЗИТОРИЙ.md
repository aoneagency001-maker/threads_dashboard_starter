# Инструкция по скачиванию репозитория на локальный компьютер

## Способ 1: Клонирование через Git (рекомендуется)

Выполните на вашем локальном компьютере:

```bash
git clone https://github.com/aoneagency001-maker/threads_dashboard_starter.git
cd threads_dashboard_starter
```

Если нужна конкретная ветка:
```bash
git clone -b cursor/placeholder-task-96d6 https://github.com/aoneagency001-maker/threads_dashboard_starter.git
```

## Способ 2: Скачать архив

Архив проекта создан и находится в `/tmp/threads_dashboard_starter.tar.gz`

Чтобы скачать архив:
1. Используйте функцию скачивания файлов в Cursor
2. Или скопируйте файл из `/tmp/threads_dashboard_starter.tar.gz`

После скачивания распакуйте архив:
```bash
tar -xzf threads_dashboard_starter.tar.gz
cd threads_dashboard_starter
```

## Установка зависимостей

После клонирования/распаковки установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка окружения

Создайте файл `.env` на основе `env_template.txt` и заполните необходимые переменные окружения.

## Важно

- Файл `.env` с секретными данными не включен в архив для безопасности
- После клонирования создайте свой `.env` файл
- Проверьте файл `API_KEYS_AND_CREDENTIALS.md` для информации о необходимых ключах
